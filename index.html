测试提示

1. 确保你的 CloudFlare Worker 已部署并正常运行

2. 测试连接时，系统会尝试访问指定的 API 地址

3. 如果测试失败，请检查你的网络连接和配置信息

Worker 部署说明

1. 将 _worker.js 文件部署到 CloudFlare Pages

2. 在 CloudFlare Pages 中绑定 KV 命名空间，变量名为 webhome

3. 生成一个 UUID 作为 SECRET_KEY 并更新到 _worker.js 文件中

4. 部署完成后，使用生成的 URL 作为 API 地址
